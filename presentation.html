🧠 Core Technology

What is a Radix Tree?

Compresses chains of single-child nodes into compact edge labels — fewer nodes, less memory, same O(L) search speed.

Standard Trie Uncompressed

root→m→u→m→[END]
         →b→a→i→[END]
         →b→l→e→[END]

⚠ 12 nodes for 3 words

Radix Tree Compressed ✓

root→"mum"→[END]
        →"bai"→[END]
        →"ble"→[END]

✅ 4 nodes — 70% fewer!

⚙️ Algorithm Detail

Edge Splitting Mechanics

When inserting a word, if a partial match occurs on an edge, the edge is split by introducing a branch node.

// 1. Find matching prefix length (j) on edge
let j = 0;
while (j < edge.length && edge[j] === word[i + j]) j++;

// 2. If partial match (j < edge.length), split!
const splitNode = new RadixNode();
const oldSuffix = edge.slice(j);
splitNode.children[oldSuffix[0]] = [oldSuffix, childNode];

// 3. Insert new suffix branch
const newSuffix = word.slice(i + j);
if (newSuffix) {
  const leafNode = new RadixNode();
  leafNode.isEnd = true;
  splitNode.children[newSuffix[0]] = [newSuffix, leafNode];
} else {
  splitNode.isEnd = true;
}

// 4. Update parent edge with shared prefix
parentNode.children[char] = [edge.slice(0, j), splitNode];

Handles dynamic tree balancing automatically during dataset loading.

📐 Math & Audio

Layout & Web Audio Synthesis

Recursive coordinate plotting and runtime frequency synthesis keep the project lightweight and fast.

📐

Trie Layout Mathematics

Calculates SVG node coordinates recursively using DFS. Leaf nodes are spaced evenly horizontally, and parent nodes are centered directly above the average of their children's positions.

🔊

Web Audio Sound Synthesis

Uses native browser `AudioContext` and oscillator nodes (sine & triangle waves) to synthesize feedback sounds (celebration chimes, error buzzes) at runtime with 0 audio files.

🗄️ Dataset Pipeline

How the City Database Was Built

The 15,000-city dataset was prepared from the raw GeoNames geographical database for India.

1

Download & Extract

Fetched IN.zip from GeoNames — all Indian geographical features.

2

Populated Place Filtering

Kept only Class P records (cities & towns). Removed lakes, monuments, admin regions.

3

Heuristic Sorting

State & national capitals first, then sorted by descending census population.

4

Clean & Deduplicate

Normalized to lowercase, removed punctuation, extracted top 15,000 unique city names.

Source: geonames.org • Feature Class P • Country IN • CC BY 4.0
